Las etiquetas semánticas ayudan a definir la estructura del documento y permiten

que las páginas web sean mejor indexadas por los buscadores. Una etiqueta se califica

como semántica si tiene que ver el significado, es decir, si nos informa sobre lo que

trata su contenido.

CSS grid layout o CSS grid es una técnica de las Hojas de Estilo en Cascada que

permite los desarrolladores web crear diseños complejos y adaptables con mayor

facilidad en todos los navegadores.
